Organizer Guide

As an organizer, you can create, start, and end tournaments. Tournaments will take place in the channel where the tournament is created, and all commands must be sent in that channel.

!to [add/remove] @role
(Server administrators only)
Add or remove a role that functions as a tournament organizer. Only server admins and any roles identified by this command can use the commands below.

!new [options]
Create a new tournament. It is strongly suggested to use the generator to create a command for a new tournament.

!start
Starts the tournament.

!list
Get a count and list of all active players. List only prints if the tournament has not started.

!json
Get a .json of the tournament. Useful for filing bug reports.

!end
Ends a tournament.

!join @player [username]
Add another player to the tournament. Username is only necessary if the tournament is using ratings from chess.com or lichess.org.

!result w-l[-d] R#M#
Enter a result for a specific match denoted by round number and match number. The result is for player one.
If the result is 0-0, then the match is reset.

!quit @player
Remove another player from the tournament. If the tournament has started, they will be dropped.
