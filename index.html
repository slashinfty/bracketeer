Bracketeer

A Discord bot that will run tournaments in your server.

Invite to Server View Live Tournaments

What Can It Do?
  • Run single and double elimination tournaments, with or without initial seeding
  • Run Swiss, round-robin, and double round-robin tournaments with elimination playoffs
  • Run chess tournaments with Dutch pairings (a variant of Swiss that balances colors) and get ratings from chess.com and lichess
  • Display pairings and standings online in real-time, and post new matches in Discord as they become active.
