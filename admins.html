Admin Guide

As an administrator, you can create, start, and end tournaments. Tournaments will take place in the channel where the tournament is created, and all commands must be sent in that channel.

!new [options]
Create a new tournament. It is strongly suggested to use the generator to create a command for a new tournament.

!start
Starts the tournament.

!list
Get a count and list of all active players. List only prints if the tournament has not started.

!end
Ends a tournament.

!result w-l[-d] @player
Edit the most recent match of a player. Wins correspond to the tagged player. Draws are only required if greater than 0.
If the result is 0-0, then the match is reset.

!quit @player
Remove another player from the tournament. If the tournament has started, they will be dropped.
