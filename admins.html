Admin Guide

As an administrator, you can create, start, and end tournaments. Tournaments will take place in the channel where the tournament is created, and all commands must be sent in that channel.

!new [options]
Create a new tournament. It is strongly suggested to use the generator to create a command for a new tournament.

!start
Starts the tournament.

!list
Get a list of all active players.

!end
Ends a tournament.

Admins also have the ability to remove other users (same command as users, but also tagging the user at the end), and overwrite existing results (by tagging the player at the end who's score is listed first - this is limited to editing the most recent match of the tagged player).
